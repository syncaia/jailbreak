# কিন্ডল ডিস্ক ফিলার ইউটিলিটি (Kindle Disk Filler Utility)

এই ইউটিলিটিটি আপনার কিন্ডল (বা যেকোনো ডিভাইসের) স্টোরেজ স্পেস পূর্ণ করতে সাহায্য করে, যা স্বয়ংক্রিয় আপডেট রোধ করার জন্য বিশেষ করে জেলব্রেক (jailbreak) এর প্রস্তুতির সময় দারুণ কার্যকর। এটি উইন্ডোজ (PowerShell) এবং লিনাক্স/ম্যাকওএস (Bash) এর জন্য উপলব্ধ।

**গুরুত্বপূর্ণ:** যখন আপনার কিন্ডল ইউএসবি (USB) দিয়ে সংযুক্ত থাকে, তখন `Filler.ps1` বা `Filler.sh` স্ক্রিপ্টটি আপনার কিন্ডলের রুট (root) ডিরেক্টরিতে কপি করুন। রুট ডিরেক্টরি হলো প্রধান ফোল্ডার যা আপনি আপনার কম্পিউটারে কিন্ডল ড্রাইভ খুললে দেখতে পান (কোনো সাবফোল্ডারের ভেতরে নয়)।

## কীভাবে ব্যবহার করবেন (Usage)

### উইন্ডোজ (PowerShell)
**টিপস:** দ্রুত সঠিক ফোল্ডারে পাওয়ারশেল (PowerShell) উইন্ডো খুলতে:
- ফাইল এক্সপ্লোরারে, `Filler.ps1` থাকা ফোল্ডারটিতে যান।
- `Shift` চেপে ধরে ফোল্ডারের ব্যাকগ্রাউন্ডে রাইট-ক্লিক করুন, এরপর **"Open PowerShell window here"** বা **"Open command window here"** নির্বাচন করুন।
- অথবা, অ্যাড্রেস বারে ক্লিক করে `powershell` টাইপ করুন এবং Enter চাপুন।

1. `Filler.ps1` থাকা ফোল্ডারে পাওয়ারশেল খুলুন।
2. যদি আপনি কোনো execution policy সম্পর্কিত এরর পান, তবে আপনি এটি রান করে তা বাইপাস করতে পারেন:
   ```powershell
   powershell -ExecutionPolicy Bypass -File .\Filler.ps1
   ```
   অথবা, শুধু বর্তমান সেশনের জন্য:
   ```powershell
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   .\Filler.ps1
   ```
3. কতোটুকু খালি জায়গা (free space) রাখতে চান, তা নির্বাচন করতে স্ক্রিনের নির্দেশাবলী অনুসরণ করুন।
4. স্ক্রিপ্টটি `fill_disk` ফোল্ডারের ভেতরে ফাইল তৈরি করতে থাকবে, যতক্ষণ না আপনার কাঙ্ক্ষিত খালি জায়গা অবশিষ্ট থাকে।

### লিনাক্স / ম্যাকওএস (Bash)
1. `Filler.sh` থাকা ফোল্ডারে একটি টার্মিনাল খুলুন।
2. স্ক্রিপ্টটিকে এক্সিকিউটেবল করুন (যদি প্রয়োজন হয়):
   ```bash
   chmod +x Filler.sh
   ```
3. স্ক্রিপ্টটি রান করুন:
   ```bash
   ./Filler.sh
   ```
4. কতোটুকু খালি জায়গা (free space) রাখতে চান, তা নির্বাচন করতে স্ক্রিনের নির্দেশাবলী অনুসরণ করুন।
5. স্ক্রিপ্টটি `fill_disk` ফোল্ডারের ভেতরে ফাইল তৈরি করতে থাকবে, যতক্ষণ না আপনার কাঙ্ক্ষিত খালি জায়গা অবশিষ্ট থাকে।

## নোটস
- আপডেট কার্যকরভাবে ব্লক করার জন্য শুধুমাত্র ২০-৫০ মেগাবাইট (20-50 MB) খালি জায়গা রাখার পরামর্শ দেওয়া হচ্ছে।
- আবার জায়গা খালি করার জন্য, জেলব্রেক প্রক্রিয়া শেষ হওয়ার পরে সহজভাবে `fill_disk` ফোল্ডারটি ডিলিট করে দিন।

## জেলব্রেকের পর: স্টোরেজ খালি করা
জেলব্রেক প্রক্রিয়া সম্পন্ন করার পর, আপনি নিরাপদে `fill_disk` ফোল্ডারটি মুছে ফেলে স্টোরেজ স্পেস পুনরুদ্ধার করতে পারেন। আপনি চাইলে শুধুমাত্র কিছু ফাইলও মুছে ফেলতে পারেন যদি আপনি ডিস্কটি আরও কিছু সময়ের জন্য প্রায় পূর্ণ রাখতে চান।

- **উইন্ডোজ:**
  1. ফাইল এক্সপ্লোরার খুলুন এবং `fill_disk` থাকা ফোল্ডারে যান।
  2. `fill_disk` ফোল্ডারটি ডিলিট করুন, অথবা এর ভেতরের নির্দিষ্ট কিছু ফাইল মুছে ফেলুন।

- **লিনাক্স / ম্যাকওএস:**
  1. `fill_disk` থাকা ফোল্ডারে একটি টার্মিনাল খুলুন।
  2. রান করুন:
     ```bash
     rm -rf fill_disk
     ```
     অথবা প্রয়োজন অনুযায়ী নির্দিষ্ট কিছু ফাইল মুছে ফেলুন।

এটি আপনার ডিভাইসের উপলব্ধ স্টোরেজ স্পেস ফিরিয়ে আনবে।
